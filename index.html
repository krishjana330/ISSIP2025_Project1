  • Robust and low complexity filter design
  • Signal reconstruction
  • Filter bank theory and wavelets

Statistical Signal Processing

  • Adaptive filtering
  • Learning algorithms for neural networks
  • Hybrid soft computing techniques
  • Spectrum estimation and modeling
  • Sensor array processing with applications in sonar and radar

Image Processing

  • Image restoration
  • Image compression
  • Image segmentation
  • Image quality evaluation
  • Computer vision and medical imaging
  • Character recognition
  • Remote Sensing Imagery

Speech Processing

Video Processing

Special Session

Special Session Format

Special sessions complement the regular program of the conference and provide a sample of the state-of-the-art research in both theoretical and application areas.

The aim of special sessions is to provide a focused discussion on innovative topics. A special session proposal should include the session title, a brief description of the scope and motivation, biographic and contact information of the organizers.

Each special session should include at least 4 papers. Once a special session proposal is accepted, the organizers will be responsible for the advertisement, paper reviewing and paper collection.

Accepted Special Session

Smart Finances: Web-Based Systems for Personalized Financial Management

Organizers: Dr. Michael Johnson, London Metropolitan University, UK

Description: This special session focuses on innovative web-based systems for personal and corporate financial management, exploring the intersection of financial technology, data analytics, and user experience design.

Advanced Image Processing Techniques for Medical Applications

Organizers: Dr. Sarah Chen, Stanford University, USA

Description: This session explores cutting-edge image processing algorithms and their applications in medical diagnostics, treatment planning, and healthcare management.
